## Website Sedang Dalam Pengerjaan 🚧

Halo, Hero!

Website resmi **Tales Hero Indonesia** saat ini sedang dalam tahap pengembangan aktif. Tim kami bekerja keras untuk menyiapkan pengalaman terbaik sebelum peluncuran resmi.

### Apa yang Sedang Kami Siapkan

- Halaman informasi karakter dan hero
- Sistem pendaftaran akun pemain
- Halaman download client game
- Papan peringkat (leaderboard)
- Dan masih banyak lagi!

### Pantau Terus

Ikuti perkembangan terbaru melalui:

- Halaman **News** ini untuk pengumuman resmi
- Server **Discord** kami untuk diskusi komunitas
- Email **support@taleshero.web.id** untuk pertanyaan

Terima kasih atas kesabarannya. Sampai jumpa saat peluncuran! ⚔️
